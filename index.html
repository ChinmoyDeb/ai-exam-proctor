About this Proctoring Dashboard

This dashboard is a browser-first Autonomous AI Exam Proctoring demonstration. It combines live face orientation tracking with object detection to identify potential exam integrity violations and captures evidence snapshots for review.

What this project does

The system monitors a candidate session in real time using the webcam. It detects:

  • Face absence or a candidate looking away from the screen
  • Prohibited items such as phone or secondary laptop displays
  • A live compliance feed with evidence snapshots and event exports

How to test it

  1. Click Start Live Session and allow camera access.
  2. Position your face clearly in the webcam viewport.
  3. Look away, leave the frame, or present a phone/laptop to trigger alerts.
  4. Watch the compliance feed update with incidents and downloadable images.

Export and review output

Use Export ZIP to download all registered violation snapshots. Use Purge System Logs to reset the feed and begin a fresh test session.
